# Logistic Regression – Ad Click Prediction

This project uses logistic regression to predict whether a user will click on an advertisement based on their behavior and demographic data.

## Dataset Features
- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Male
- Clicked on Ad (Target)

## Steps
1. Data exploration and visualization  
2. Feature selection  
3. Train-test split  
4. Model training using Logistic Regression  
5. Model evaluation using classification report  

## Model Performance
The model achieved around 91% accuracy on the test data.

## Tools Used
- Python  
- Pandas  
- Seaborn  
- Scikit-learn
